# Data Import Templates

## Loan Repayment bulk import

1. Desk → Data Import → New
2. Select DocType: **Loan Repayment**
3. Download template; fill required fields:

| Column | Example |
|--------|---------|
| against_loan | LON-2026-00001 |
| applicant_type | Customer |
| applicant | Test Borrower One |
| company | LMS Demo Co |
| posting_date | 2026-05-20 |
| amount_paid | 1500 |

4. Import and submit rows from Loan Repayment list.

## Customer branch assignment

Import or update **Customer** with `custom_lms_branch` (Cost Center name).
